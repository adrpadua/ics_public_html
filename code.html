Hi, I code and design UI's for fun (and money)

My coding adventure started while living in an apartment in Hollywood as a musician. Now I'm graduating with a C.S. degree in June at UC Irvine!

Here's some stuff I've worked on:

D&D 5th Edition API

One winter I wanted to create an app that allowed one-click character creation for the tabletop RPG, Dungeons & Dragons. The main thing that held me back was that there was no JSON database to serve game data. So I modeled the data into JSON classes and served them over an API through Heroku.

Pathfinder RPG UI Concept

Back when I started really getting into code, I started out with iOS Applications. My first idea for an iOS app was a character generator for the Pathfinder RPG. At the time I hardly knew anything about actual coding, so I just created the front-end prototype on Sketch.
